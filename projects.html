Compiler C++

Build a compiler for an simplified imperative language ASL. It incorporates a grammar for the language, a semantic check module to perform the syntactic analysis and a code generation module that generates low-level code for the input program.

Developed with: C++, ANTLR4

Collaborative project to develop a program which allows to detect of the football team that appears on the image. First step is create a model for each team and then get all fragments of the image to compare theirs color histograms with model.
